# Demo Guide

## Rust Demo

```bash
cd rust-public-examples
cargo run
```

Expected output:

1. Receives fake stream events.
2. Parses sanitized events.
3. Produces fake demo scores.
4. Selects a stub route.
5. Applies a stub policy.
6. Dry-runs a send.
7. Watches fake outcomes.

## Python Demo

```bash
python python-public-examples/fake_signal_generator.py
python python-public-examples/demo_dashboard_data.py
```
