# Fluid

Fluid is the public dashboard and analytics layer for the Liquiphy + Fluid submission.

Links:

- Site: https://trifluid.com
- Repository: https://github.com/liquiphyofficial-ux/Fluid

Fluid shows the user-facing side of the system: dashboards, public reports, token views, watchlists, settings, and analytics surfaces. Private trading logic remains outside the Fluid public layer.
